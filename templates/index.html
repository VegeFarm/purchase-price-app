1. 거래명세서 링크 입력

링크의 HTML을 가져온 뒤, hidden input name="statement" 안의 JSON을 파싱합니다.

2. 미등록상품 재처리

미등록상품의 처리대기/사유 상태 행을 품목환산표 기준으로 다시 확인하고, 가능하면 매입기록에 입력합니다.

3. 상품명 일괄변경

상품명변경 시트의 기존상품명 → 변경상품명 기준으로 매입기록 시트의 상품명만 변경합니다.

4. 매입기록 삭제

서식과 E:F 수식을 보호하기 위해 매입기록 시트 A:D 값만 비웁니다. 전표관리는 자동 삭제하지 않습니다.

이 프로그램은 템플릿의 시트/컬럼을 검증만 하고, 필요한 값 입력·값 삭제 외에는 서식/드롭다운/조건부서식/수식/열 너비/셀 병합을 수정하지 않습니다.
